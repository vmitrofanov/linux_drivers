# Repository summary
##### bb_eternet 
100M ethernet Linux driver for Beagle Bone Black. Still in progress. In this driver I use original PHY driver from mainline. I have ported MDIO. I have made simple transferring and receiving. It is left to add multiple queue support and tidy up in the dirver.

##### fm24_eeprom
Linux driver for a popular EEPROM. Finished.

##### pci_board_x100
Linux driver for a PCI board with a Ethernet switch. It is just a wripper for suitable access to board resources. Finished.

##### af_xdp
Now I'm in process of publishing my own AF_XDP driver support for 1G Ethernet. So it is empty now.
